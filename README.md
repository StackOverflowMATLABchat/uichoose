# UICHOOSER

`uichooser` opens a dialog box that allows the user to select folder(s) and/or file(s) in a single UI. This is designed as a generic alternative to MATLAB's more specific UI selection tools: [`uigetdir`](https://www.mathworks.com/help/matlab/ref/uigetdir.html) and [`uigetfile`](https://www.mathworks.com/help/matlab/ref/uigetfile.html)

This tool utilizes MATLAB's [`string`](https://www.mathworks.com/help/matlab/ref/string.html) objects, introduced in R2016b
